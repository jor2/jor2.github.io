Second Year Project
Java Web Appl using Play Framework

Summary

I did this project with two other peers from my class in ITT.... We created a dynamically generating web application that connected to Oracle db's and was built on the play framework platform. We started by creating all our use cases, schema etc so that we could have a good idea of what we were going to build before we built it. We created a video games website that would allow a user to buy games, build up loyalty rewards etc. The admin could update the stock, remove or add games and reviews.

Functionality:
- Reviews
- Loyalty Points
- Add, Update and Delete
- Add Credit Cards to Account

We had a static version of the site available to us prior to starting the app from our Previous web dev class. We used this template to create the games site. We familiarized ourselves a lot more with play, sql, java and scala. Firstly, we created a static HTML site before converting the site to Scala language within Play Framework. Each week we would meet with our Project
